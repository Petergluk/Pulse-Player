# PulsePlayer 🏃‍♂️🎧

**PulsePlayer** — это умный музыкальный плеер для тренировок, который автоматически подбирает музыку под ваш текущий пульс или темп шагов. Идеально подходит для бега, кардио-тренировок, велоспорта и просто активных прогулок.

## 🌟 Основные возможности

### 🎛 Три режима работы
* **Фикс (Ручной режим):** Установите желаемый BPM (ударов в минуту) вручную с помощью кнопок `+` и `-`. Плеер будет играть музыку строго в заданном темпе.
* **Пульс (Heart Rate):** Подключите любой Bluetooth-пульсометр (нагрудный датчик или фитнес-часы с поддержкой передачи пульса по Bluetooth). Плеер будет считывать ваш пульс в реальном времени и подбирать треки, ритм которых совпадает с частотой вашего сердцебиения.
* **Шаги (Pedometer):** Использует датчики движения (акселерометр) вашего устройства для подсчета шагов. Музыка подстраивается под темп вашей ходьбы или бега.

### 🎵 Источники музыки
* **Jamendo API:** Огромная библиотека бесплатной музыки. Выберите любимый жанр (Pop, Rock, Electronic, Lo-Fi и др.), и плеер сам найдет подходящие треки в интернете.
* **Локальные файлы (Мои файлы):** Слушайте свою собственную музыку! Плеер умеет автоматически определять BPM (темп) прямо из названий ваших файлов. 
  * *Поддерживаемые форматы названий файлов:* 
    * С явным указанием: `Track 130bpm.mp3`, `Song 125 bpm.wav`
    * С префиксом: `11_080-Song.mp3`, `095_Track.mp3`
    * Просто число от 60 до 220: `MyTrack - 125 - Mix.mp3`

### 🧠 Умные алгоритмы и настройки (⚙️)
* **Кроссфейд (Плавный переход):** Настройте плавное перетекание одной песни в другую (от 0 до 10 секунд). Треки больше не будут обрываться резко, создавая эффект непрерывного диджейского микса.
* **Окно усреднения темпа:** Защита от "дерганого" пульса или сбоев шагомера. Плеер высчитывает средний темп за заданное время (от 10 до 120 секунд) и меняет трек только при стабильном изменении ритма тренировки.
* **Слушать трек до конца:** Опция, запрещающая прерывать текущую песню на середине. Даже если ваш пульс изменился, плеер доиграет красивый трек до конца, а уже следующий подберет под новый темп.
* **Режим заминки (Cooldown):** Установите порог максимального пульса (например, 180 BPM). Если вы превысите его, плеер включит режим восстановления и будет плавно снижать темп музыки в течение заданного времени (например, 1 минуты), помогая сердцу успокоиться после пиковой нагрузки.

## ⚙️ Установка и запуск (для разработчиков)

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Создайте файл `.env` в корне проекта (можете использовать шаблон `.env.example`) и добавьте ваш Jamendo Client ID:
   ```env
   VITE_JAMENDO_CLIENT_ID=ваш_ключ_от_jamendo
   ```
   *Ключ можно получить бесплатно на [developer.jamendo.com](https://developer.jamendo.com/)*.
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

## 📱 Особенности использования локальных файлов
Музыка, которую вы выбираете через кнопку "Мои файлы", **никуда не загружается в интернет**. Она обрабатывается локально прямо в памяти вашего браузера, что гарантирует мгновенное переключение треков и полную приватность. 
*Примечание для смартфонов:* Из-за ограничений мобильных ОС (iOS/Android) выбор папки целиком может быть недоступен. В таком случае просто выделите сразу несколько аудиофайлов в открывшемся окне выбора файлов.
